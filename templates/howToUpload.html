How to upload a workout

Step 1: Get a workout sheet template

On the "home" screen, select "Blank sheet" under the "Upload workout" header. This will download an Excel file named "workout_d_m_y.xlsx" (where d, m, y are the day, month, year respectively). Open the file.

Step 2: Fill out the template

Filling out the template correctly is important for irgo to be able to read it. Times and distances need to be formatted correctly, and the text 'irgo' and 'Piece:' in the header row must not be changed.

First, fill out information about the workout. Make sure the date (cell A2) is correct. Next, in the header row, edit the cells with '(XXXX | mm:ss) with the pieces. Distance pieces should be entered as the number of meters, and time pieces should be minutes : seconds. E.g., a 6k goes in as '6000', a 30-minute goes in as '30:00'.

Next, fill out the athlete names. The roster of athletes is filled out in the "roster" sheet, with which the "First" field (column A) is filled with a VLOOKUP. Enter the last names, and make sure that a first name is filled in. If it returns an error, check the spelling or that the athlete is on the roster.

Finally, fill out the scores. Like the pieces, formatting is important. For distance pieces, you must enter the score as a time (format: mm:ss). For timed pieces, you must enter a distance (format: dddd). The upload will not work if these are mismatched.

If an athlete didn't complete a piece, write "DNF" in the cell

Step 3: Upload the workout

Go back to irgo, to the home screen. Select "Upload Sheet", where you will be prompted to choose a file from your computer. Choose the file you filled out. If it was successful, you will be redirected to the page showing that workout. Congrats, you've uploaded a workout.
